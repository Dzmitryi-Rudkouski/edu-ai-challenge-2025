## Пример 4: Обработка ошибок

### Команда с неверным API ключом:
```bash
python main.py "Test" --api-key "invalid-key"
```

### Вывод:
```
[bold red]Ошибка API ключа: Invalid API key[/bold red]
Проверьте ваш API ключ и попробуйте снова.
```

### Команда без обязательных параметров:
```bash
python main.py
```

### Вывод:
```
[bold red]Ошибка: Необходимо указать название сервиса или описание[/bold red]
Примеры использования:
  python main.py 'Spotify'
  python main.py 'Новый сервис' --description 'Платформа для создания онлайн-курсов'
```
